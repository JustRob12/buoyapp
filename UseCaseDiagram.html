Buoy Monitoring System - Use Case Diagram

System Overview

The Buoy Monitoring System is a mobile application that monitors and displays real-time water quality data from multiple buoys. The system fetches data from an external API and provides features for data visualization, analysis, and export.

Use Case Relationships:

  • Include (<>): Required functionality that must be performed as part of the main use case
  • Extend (<>): Optional functionality that can be performed to enhance the main use case

Main Use Cases

  • View Dashboard - Main interface
  • View Data Table - Historical data
  • View Graphs - Data visualization
  • View Map - Geographic view
  • Manage Settings - App configuration

Include Use Cases (Required)

  • Select Buoy - Choose specific buoy
  • Refresh Data - Update information
  • Download CSV - Export data
  • View Charts - Display graphs
  • Configure App - Set preferences

Extend Use Cases (Optional)

  • Notifications - Push alerts
  • Offline Mode - Cached data
